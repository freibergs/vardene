vardene

Latvian morphological analysis — Python port of the LU MII engine

Returns the top reading per token after CRF + classifier + Viterbi disambiguation.

Generates every inflected form. Verb lemmas can produce 500+ forms across tenses, persons, moods, and participles.

HTTP API

All endpoints return JSON. Tokenisation in /analyzesentence and /morphotagger uses a lightweight whitespace + punctuation regex; a port of Splitting.java is pending.

MethodEndpointDescription
GET/api/analyze/<word>Single-word analysis (LV attributes)
GET/api/analyze/en/<word>Same with English attribute names
GET/api/analyzesentence/<query>Per-token analysis with sentence context
GET/api/morphotagger/<query>Sentence-level disambiguation (best reading per token)
GET/api/tokenize/<query>Tokenise a sentence
POST/api/tokenizeSame with body {"text": "..."}
GET/api/v1/inflections/<lemma>All inflectional forms of a lemma
GET/api/v1/inflections/<lemma>?paradigm=<name>With explicit paradigm
GET/api/v1/inflections/<lemma>?paradigm=&stem1=&stem2=&stem3=Verb-1 with explicit stems
GET/api/inflect/json/<lemma>Format-selectable inflection (JSON only at the moment)
GET/api/inflect/json/<lang>/<lemma>With language filter (lv/ltg)
GET/api/healthLiveness probe

Pending (501 Not Implemented)

  • /api/verbs, /api/neverbs — external valency annotation tool
  • /api/suitable_paradigm — paradigm suitability scorer
  • /api/inflect_people — person-name inflector
